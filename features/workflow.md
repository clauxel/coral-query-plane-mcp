# Workflow

Coral Query Plane is a hosted remote MCP for AI SDK data query MCP.

## Repeatable Flow

1. Open Coral Query Plane and select the buyer plan.
2. Create or request a bearer token from the hosted product.
3. Add https://coralquery.clauxel.com/mcp to a compatible MCP client.
4. Run tools/list, then call source_list with public-safe sample data.
5. Save the returned receipt or export for human review.

## Output Mindset

The useful artifact is not a marketing claim. It is evidence that a reviewer can inspect, export, and compare later.
